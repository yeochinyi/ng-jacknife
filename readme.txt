Get started....

# Install Karma:
https://karma-runner.github.io/1.0/intro/installation.html
1) npm install karma --save-dev

# Install plugins that your project needs:
2) npm install karma-jasmine karma-chrome-launcher karma-phantomjs-launcher --save-dev

# Install phantomjs
3) http://phantomjs.org/quick-start.html

4) npm install

5) bower install

6) karma start
